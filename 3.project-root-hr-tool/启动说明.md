## 🚀 立即开始使用

### 1. 打开应用

在浏览器中访问：**http://localhost:3000**

### 2. 界面说明

应用分为三列布局：

**左列 - 上传和问题列表**
- 📤 **上传文档**：拖拽或点击上传文件
- 📋 **发现的问题**：显示所有检测到的问题
  - 可按类型筛选（语法、格式、一致性、结构、内容）
  - 可按严重程度筛选（关键、高、中、低）
  - 支持搜索功能

**中列 - 文档预览和问题详情**
- 📄 **文档预览**：显示上传的文档内容
- 🔍 **问题详情**：显示选中问题的详细信息
  - 原始文本
  - 上下文
  - 修复建议
  - 置信度
  - 可点击"应用修复"或"手动编辑"

**右列 - 修复预览和下载**
- 🔄 **修复预览**：对比修复前后的文本
- 💾 **下载管理**：下载原始或修复后的文档

## 📝 使用流程

### 步骤 1：上传文档
1. 在左上角"上传文档"区域
2. 拖拽文件或点击选择文件
3. 支持格式：PDF、Word (.docx)、Excel (.xlsx)、文本 (.txt)
4. 文件大小限制：50MB

### 步骤 2：等待分析
- 系统自动解析文档
- 调用 DeepSeek AI 进行智能分析
- 分析完成后显示所有问题

### 步骤 3：查看问题
- 在"发现的问题"列表中浏览所有问题
- 点击任意问题查看详情
- 问题按严重程度排序（关键 > 高 > 中 > 低）

### 步骤 4：应用修复
- 在问题详情中点击"应用修复"
- 或选择多个问题批量修复
- 在"修复预览"中查看修复效果

### 步骤 5：下载文档
- 在"下载管理"面板选择格式
- 点击"下载"按钮
- 选择"原始文档"或"修复后文档"

## 🐛 常见问题

### Q: 上传后没有反应？
A: 请检查：
- 文件格式是否支持
- 文件大小是否超过 50MB
- 查看浏览器控制台是否有错误信息

### Q: 分析失败？
A: 可能原因：
- DeepSeek API 调用失败
- 网络连接问题
- 文档内容无法解析

### Q: 如何查看详细日志？
A: 后端日志位置：`backend/logs/combined.log`

## 📊 API 端点

如果需要直接调用 API：

```bash
# 健康检查
curl http://localhost:3001/health

# 上传文档
curl -X POST http://localhost:3001/api/documents/upload \
  -F "file=@your-document.pdf"

# 分析文档
curl -X POST http://localhost:3001/api/documents/{documentId}/analyze

# 获取分析结果
curl http://localhost:3001/api/documents/{documentId}/analysis/{analysisId}

# 下载文档
curl http://localhost:3001/api/documents/{documentId}/download?format=fixed
```
