# 🧪 测试 PDF 导出功能

## 快速测试步骤

### 1. 刷新应用
```
在浏览器中按 Ctrl + F5 强制刷新
```

### 2. 生成报告
- 上传答题卡图片
- 等待 AI 分析完成
- 查看生成的报告

### 3. 测试导出
- 点击右上角"导出"按钮
- 在弹出对话框中点击"确定"（选择 PDF 格式）
- 浏览器会打开打印对话框

### 4. 保存 PDF
在打印对话框中：
- **目标打印机**：选择"另存为 PDF"或"Microsoft Print to PDF"
- **页面设置**：
  - 纸张大小：A4
  - 边距：默认
  - 取消勾选"页眉和页脚"
- 点击"保存"
- 选择保存位置
- 完成！

## 故障排除

### 问题 1：打印对话框没有出现
**解决方案**：
- 检查浏览器是否阻止了弹窗
- 允许弹窗后重试

### 问题 2：PDF 样式不正确
**解决方案**：
- 确保使用 Chrome/Edge 浏览器
- 清除浏览器缓存后重试

### 问题 3：导出按钮无响应
**解决方案**：
- 按 F12 打开开发者工具
- 查看 Console 是否有错误
- 刷新页面后重试
