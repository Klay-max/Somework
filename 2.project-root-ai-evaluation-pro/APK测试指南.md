## 测试方法（选择其中一种）

### 方法 1：直接传输到手机（最简单）✨

这是最简单的方法，不需要安装任何额外工具。

#### 步骤：

1. **找到 APK 文件**
   - 在文件资源管理器中打开：`D:\桌面文件\project-root\android\app\build\outputs\apk\release`
   - 你会看到 `app-release.apk` 文件

2. **传输到手机**
   - 方式 A：通过 USB 数据线连接手机，将 APK 复制到手机存储
   - 方式 B：通过微信/QQ 发送给自己
   - 方式 C：上传到云盘（百度网盘、阿里云盘等）后在手机下载
   - 方式 D：通过邮件发送给自己

3. **在手机上安装**
   - 在手机上找到 APK 文件
   - 点击安装
   - 如果提示"禁止安装未知应用"，需要先启用权限：
     - 设置 → 安全 → 未知来源 → 允许安装应用

### 方法 2：使用 ADB 安装（推荐给开发者）🔧

ADB 可以直接从电脑安装 APK 到手机，非常方便。

#### 安装 ADB：

**选项 A - 安装完整 Android Studio（较大）**

1. 下载 Android Studio：https://developer.android.com/studio
2. 安装后，ADB 会自动包含在内

**选项 B - 只安装 Platform Tools（推荐，较小）**
1. 下载 SDK Platform Tools：https://developer.android.com/studio/releases/platform-tools
2. 解压到任意文件夹（例如：`C:\platform-tools`）
3. 将该文件夹路径添加到系统环境变量 PATH

#### 使用 ADB 安装：

1. **启用手机的 USB 调试**
   - 设置 → 关于手机 → 连续点击"版本号"7次（启用开发者选项）
   - 设置 → 开发者选项 → USB 调试（开启）

2. **连接手机到电脑**
   - 使用 USB 数据线连接
   - 手机上会弹出"允许 USB 调试"提示，点击"允许"

3. **验证连接**
   ```bash
   adb devices
   ```
   应该显示你的设备

4. **安装 APK**
   ```bash
   adb install android\app\build\outputs\apk\release\app-release.apk
   ```

5. **如果已安装旧版本，使用覆盖安装**
   ```bash
   adb install -r android\app\build\outputs\apk\release\app-release.apk
   ```

## ⚠️ 常见问题

### 1. 安装时提示"应用未安装"
- 可能是签名冲突，卸载旧版本后重新安装
- 或使用 `adb install -r` 覆盖安装

### 2. 安装时提示"禁止安装未知应用"
- 设置 → 安全 → 未知来源 → 允许安装应用
- 或在安装时点击"设置"直接跳转

### 3. 应用闪退
- 检查手机 Android 版本（需要 Android 5.0+）
- 查看崩溃日志：`adb logcat | findstr "AndroidRuntime"`

### 4. 相机无法打开
- 检查相机权限是否授予
- 设置 → 应用 → 你的应用 → 权限 → 相机

### 5. OCR/AI 功能不工作
- 检查网络连接
- 确认 API 密钥已正确配置
- 或切换到"模拟模式"进行测试
