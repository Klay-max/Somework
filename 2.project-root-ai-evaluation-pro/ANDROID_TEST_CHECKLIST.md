# Android 测试检查清单

## 构建前检查

- [ ] 已安装 EAS CLI (`npm install -g eas-cli`)
- [ ] 已登录 Expo 账号 (`eas login`)
- [ ] 已配置项目 (`eas build:configure`)
- [ ] 环境变量已在 Expo 配置（如果需要）

## 构建检查

- [ ] 选择构建方式（云端/本地）
- [ ] 构建成功完成
- [ ] APK 文件已下载/生成
- [ ] APK 文件大小合理（< 50MB）

## 安装检查

- [ ] 手机已启用"未知来源"安装
- [ ] APK 成功安装到手机
- [ ] 应用图标显示正常
- [ ] 应用可以正常启动

## 权限检查

- [ ] 相机权限请求正常
- [ ] 相机权限已授予
- [ ] 存储权限请求正常（如果需要）
- [ ] 网络权限正常

## 功能测试

### 1. 首页测试
- [ ] 首页正常显示
- [ ] 赛博朋克主题正确（黑色背景，青色文字）
- [ ] "启动视觉诊断"按钮可点击
- [ ] "查看历史记录"按钮可点击

### 2. 扫描页面测试
- [ ] 扫描页面正常显示
- [ ] 相机可以正常打开
- [ ] 可以拍照或选择图片
- [ ] 图片预览正常显示

### 3. OCR 识别测试
- [ ] 点击"开始扫描"后显示进度
- [ ] 进度文本正确显示（1/6, 2/6...）
- [ ] OCR 识别成功（或显示错误信息）
- [ ] 识别时间合理（< 10秒）

### 4. AI 分析测试
- [ ] 错误分析步骤显示（5/6）
- [ ] 学习路径生成步骤显示（6/6）
- [ ] AI 分析成功（或显示错误信息）
- [ ] 分析时间合理（< 15秒）

### 5. 报告页面测试
- [ ] 自动跳转到报告页面
- [ ] 核心得分正确显示
- [ ] 五维雷达图正确渲染（无 NaN 错误）
- [ ] 深度分析内容显示
- [ ] 知识点矩阵显示
- [ ] 提分路径显示
- [ ] 所有数据格式正确

### 6. 历史记录测试
- [ ] 历史记录页面正常显示
- [ ] 报告列表正确显示
- [ ] 可以点击查看历史报告
- [ ] 历史报告数据完整

### 7. 网络测试
- [ ] WiFi 环境下正常工作
- [ ] 移动数据环境下正常工作
- [ ] 网络错误时显示友好提示
- [ ] 超时错误处理正确

### 8. 性能测试
- [ ] 应用启动速度快（< 3秒）
- [ ] 页面切换流畅
- [ ] 图像处理不卡顿
- [ ] 内存使用合理（< 200MB）

### 9. 错误处理测试
- [ ] 无网络时显示错误提示
- [ ] OCR 失败时显示错误提示
- [ ] AI 超时时显示错误提示
- [ ] 图像质量差时显示提示

### 10. UI/UX 测试
- [ ] 所有文字清晰可读
- [ ] 按钮大小合适，易于点击
- [ ] 颜色对比度足够
- [ ] 加载动画流畅
- [ ] 错误提示清晰明确
