# 生成新图标指南 🎨

## 问题
当前应用图标还是墨绿色的旧图标，需要更换为新的教育风格蓝色图标。

## 最简单的解决方案

### 方法 1：使用 Figma（推荐，最快）⭐

1. **打开 Figma**（免费账号即可）
   - 访问：https://www.figma.com/

2. **创建新设计**
   - 创建 1024x1024 的画布

3. **设计图标**
   ```
   背景：蓝色渐变
   - 顶部：#4A90E2
   - 底部：#67B8F7
   
   图标元素：
   - 白色书本图标（居中）
   - 橙色 "AI" 标记（右上角）
   ```

4. **导出图标**
   - 选择画布
   - 导出为 PNG
   - 尺寸：1024x1024
   - 保存 3 个文件：
     * `icon.png` (1024x1024)
     * `adaptive-icon.png` (1024x1024)
     * `favicon.png` (48x48，缩小版本)

5. **替换文件**
   - 将生成的文件复制到 `assets/` 文件夹
   - 覆盖原有文件

---

### 方法 2：使用 Canva（最简单）

1. **打开 Canva**
   - 访问：https://www.canva.com/

2. **创建自定义尺寸**
   - 选择"自定义尺寸"
   - 输入：1024 x 1024 像素

3. **设计图标**
   - 添加渐变背景（蓝色）
   - 搜索"书本"图标，添加白色书本
   - 添加文字 "AI"（橙色）

4. **下载**
   - 点击"下载"
   - 选择 PNG 格式
   - 下载后重命名为 `icon.png`

5. **生成其他尺寸**
   - 复制设计
   - 调整尺寸为 48x48（favicon）
   - 下载并重命名为 `favicon.png`

6. **复制文件**
   - `icon.png` → `adaptive-icon.png`（复制一份）
   - 将所有文件放到 `assets/` 文件夹

---

### 方法 3：使用在线图标生成器

#### 选项 A：App Icon Generator
1. 访问：https://www.appicon.co/
2. 上传一个简单的蓝色方块图片
3. 自动生成所有尺寸
4. 下载 Android 图标包
5. 提取需要的文件到 `assets/`

#### 选项 B：Icon Kitchen
1. 访问：https://icon.kitchen/
2. 选择"Custom Image"
3. 上传你的设计
4. 选择"Adaptive Icon"
5. 下载并提取文件

## 常见问题

### Q: 图标必须是正方形吗？
A: 是的，Android 图标必须是 1024x1024 的正方形。

### Q: 可以使用透明背景吗？
A: 不建议。最好使用纯色或渐变背景。

### Q: 图标太复杂会怎样？
A: 在小尺寸下可能看不清。建议保持简洁。

### Q: 需要多少个图标文件？
A: 最少 3 个：
- icon.png (应用图标)
- adaptive-icon.png (Android 自适应图标)
- favicon.png (网页图标)

### Q: 生成后需要重新构建 APK 吗？
A: 是的，图标是打包在 APK 中的，必须重新构建。
