# Assets 目录

此目录用于存放应用的静态资源文件。

## 需要的文件

请添加以下图片文件到此目录：

1. **icon.png** - 应用图标 (1024x1024 px)
   - 赛博朋克风格
   - 建议使用青色/绿色霓虹效果

2. **splash.png** - 启动屏幕 (1242x2436 px)
   - 纯黑背景
   - 中央显示 安辅导 logo

3. **adaptive-icon.png** - Android 自适应图标 (1024x1024 px)
   - 前景图层，透明背景

4. **favicon.png** - Web 端图标 (48x48 px)
   - 简化版应用图标

## 临时方案

在开发阶段，可以使用纯色占位图片。Expo 会在缺少这些文件时使用默认图标。

## 设计建议

- 主色调：青色 (#00ffff) 和绿色 (#00ff00)
- 背景：纯黑 (#000000)
- 风格：赛博朋克、科技感、霓虹效果
