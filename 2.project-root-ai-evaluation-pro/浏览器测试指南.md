## 如果遇到问题

### 问题 1: 浏览器显示空白
**解决方案**:
1. 打开浏览器开发者工具（按 F12）
2. 查看 Console 标签页的错误信息
3. 尝试硬刷新页面（Ctrl + Shift + R）

### 问题 2: 样式显示不正常
**解决方案**:
1. 清除浏览器缓存
2. 使用无痕模式打开
3. 检查是否有 CSS 加载错误

### 问题 3: 功能无法点击
**解决方案**:
1. 检查浏览器控制台是否有 JavaScript 错误
2. 确认 Metro Bundler 没有编译错误
3. 尝试重启开发服务器

## 技术细节

### i18n 实现
项目使用自定义的轻量级 i18n 实现：
- **文件**: `lib/i18n/index.ts`
- **语言文件**: 
  - `lib/i18n/locales/zh-CN.json`
  - `lib/i18n/locales/en-US.json`
- **特性**:
  - 支持中英文切换
  - 支持变量替换
  - 支持语言持久化
  - 无外部依赖冲突
