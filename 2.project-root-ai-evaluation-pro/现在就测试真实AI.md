## 🐛 故障排除

### 问题 1: 看到"网络错误"

**解决方案**：
1. 确认 API 服务器正在运行
2. 在浏览器中按 Ctrl+F5 强制刷新
3. 清除浏览器缓存

### 问题 2: 分析内容看起来是模拟数据

**可能原因**：
1. DeepSeek API 调用失败（检查服务器日志）
2. API 密钥无效（检查 .env.local）
3. 网络问题（检查网络连接）

**解决方案**：
```bash
# 重启 API 服务器
# 按 Ctrl+C 停止当前服务器
node local-api-server.js
```

### 问题 3: OCR 识别失败

**说明**：
- 阿里云 OCR 目前有网络连接问题
- 应用会自动降级到模拟 OCR 数据
- 不影响 AI 分析功能

**临时方案**：
- 手动输入答案
- 或等待部署到 Vercel 后测试
