# 🚀 快速测试指南

## 📍 APK 位置
```
android\app\build\outputs\apk\release\app-release.apk
```

## ⚡ 最快安装方法

### 方法 1：手动传输（无需工具）
1. 复制 APK 到手机（USB/微信/QQ）
2. 在手机上点击安装
3. 允许"未知来源"权限

### 方法 2：ADB 安装（需要 ADB）
```bash
# 如果已安装 ADB，直接运行：
install-apk.bat

# 或手动执行：
adb install -r android\app\build\outputs\apk\release\app-release.apk
```

## 🧪 核心测试项（5分钟快速测试）

1. **启动测试**
   - [ ] 应用能正常打开
   - [ ] 无崩溃

2. **相机测试**
   - [ ] 打开相机
   - [ ] 拍一张照片

3. **OCR 测试**
   - [ ] 切换到"模拟模式"（设置页面）
   - [ ] 上传一张图片
   - [ ] 查看识别结果

4. **导航测试**
   - [ ] 切换各个页面
   - [ ] 查看历史记录

5. **设置测试**
   - [ ] 切换语言
   - [ ] 切换模拟模式

## ✅ 测试通过标准

- 应用能正常启动
