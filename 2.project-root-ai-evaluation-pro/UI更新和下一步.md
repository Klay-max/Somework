## ⏳ 待完成的工作

### 1. 生成新图标（重要）

由于 Windows 上 canvas 安装复杂，建议使用以下方法之一：

#### 方法 A：在线工具（最简单）

1. 打开 `icon-design.svg` 文件
2. 访问在线转换工具：
   - https://svgtopng.com/
   - https://cloudconvert.com/svg-to-png
3. 生成以下尺寸：
   - `icon.png`: 1024x1024
   - `adaptive-icon.png`: 1024x1024  
   - `favicon.png`: 48x48
4. 保存到 `assets/` 文件夹，替换现有文件

#### 方法 B：使用 Figma（推荐）

1. 在 Figma 中创建 1024x1024 画布
2. 参考 `icon-design.svg` 的设计：
   - 蓝色渐变背景 (#4A90E2 → #67B8F7)
   - 白色书本图标
   - 橙色 AI 标记
3. 导出为 PNG
4. 保存到 `assets/` 文件夹

#### 方法 C：使用 Photoshop/GIMP

1. 打开 `icon-design.svg`
2. 调整尺寸为 1024x1024
3. 导出为 PNG
4. 保存到 `assets/` 文件夹

### 2. 生成启动画面

创建 `splash.png` (1284x2778)：
- 蓝色渐变背景
- 中央放置图标
- 下方添加 "安辅导" 文字
- 副标题："AI 智能作业批改助手"

### 3. 重新构建 APK

图标生成后，需要重新构建：

```bash
# 方法 1：EAS Build（推荐）
eas build --platform android --profile preview2

# 方法 2：本地构建
npx expo prebuild --platform android --clean
cd android
.\gradlew assembleRelease --no-daemon
```

### 4. 测试 OTA 更新

新 APK 安装后：
1. 关闭应用
2. 重新打开
3. 等待 3 秒
4. 应该看到更新提示
5. 点击"立即更新"
6. 验证绿色横幅是否显示
