## 🐛 遇到问题？

### 常见问题快速解决：

**问题 1：无法安装**
- 启用"允许安装未知来源应用"
- 卸载旧版本后重新安装

**问题 2：应用闪退**
- 检查 Android 版本是否 ≥ 5.0
- 清除应用数据后重试

**问题 3：相机无法打开**
- 授予相机权限
- 检查其他应用是否占用相机

**问题 4：OCR/AI 不工作**
- 切换到"模拟模式"测试
- 检查网络连接
- 确认 API 密钥配置

更多问题请查看 `APK测试指南.md` 的"常见问题"部分。
