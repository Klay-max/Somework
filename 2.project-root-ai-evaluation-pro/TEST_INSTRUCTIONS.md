# 测试说明 - 3 步搞定

## 第 1 步：启动服务器

**在项目文件夹中双击这个文件：**
```
start-local-dev.bat
```

会打开 2 个黑色窗口，**不要关闭！**

等待看到：
- 窗口 1：`✓ 本地 API 服务器运行在 http://localhost:3001`
- 窗口 2：`› Press w │ open web`

**在窗口 2 中按键盘 `w` 键**

## 第 2 步：测试扫描

浏览器会自动打开，然后：

1. 点击 **"启动视觉诊断"**
2. 点击虚线框，选择任意图片
3. 点击 **"开始扫描"**
4. 等待处理（显示 1/6 到 6/6）
5. 自动跳转到报告页面

## 第 3 步：告诉我结果

### 如果成功了 ✓

告诉我："测试成功！"

我们就可以部署到生产环境了。

### 如果失败了 ✗

请截图给我看：
1. 浏览器地址栏
2. 页面显示的内容
3. 按 F12 打开控制台，截图 Console 标签的内容

## 重要提示

### ⚠ 如果显示 "Cannot find module"

说明你不在项目目录下！

**解决方法：**
1. 打开文件资源管理器
2. 进入：`D:\桌面文件\project-root\2.project-root-ai-evaluation-pro`
3. 在文件夹空白处，按住 Shift 键，右键点击
4. 选择 "在此处打开 PowerShell 窗口"
5. 再次双击 `start-local-dev.bat`

### ⚠ 如果浏览器没有自动打开

手动访问：`http://localhost:8081`

### ⚠ 如果显示 "Not found"

这是我们要解决的问题！

请：
1. 按 F12 打开浏览器控制台
2. 截图 Console 标签的内容
3. 截图浏览器地址栏
4. 发给我
